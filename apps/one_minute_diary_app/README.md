## Support Page (GitHub Pages)

このリポジトリには、公開用サポートページを `docs` フォルダで管理しています。

### 公開手順

1. GitHubの対象リポジトリを開く
2. `Settings` → `Pages` を開く
3. `Source` で `Deploy from a branch` を選択
4. `Branch` は `main`（または `master`）、`Folder` は `/docs` を選択
5. `Save` を押す

### 公開URL形式

`https://<username>.github.io/<repo>/`
