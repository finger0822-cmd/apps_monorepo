プライバシーポリシー

MindCapsule

収集する情報

本アプリでは、以下の情報を扱います。

  • ローカルに保存するデータ … 日記の内容・気分記録（エネルギー、集中度、疲れ、気分、眠気など）は、お使いの端末内にのみ保存されます。当方のサーバーには送信されません。
  • RevenueCat が収集する購入情報 … サブスクリプション（プレミアムプラン）の購入・復元の管理のため、RevenueCat のサービスを利用しています。購入状態やデバイス情報等が RevenueCat により収集・処理されます。詳細は RevenueCat のプライバシーポリシー をご確認ください。
  • Anthropic Claude API に送信するデータ … AI 機能をご利用になる場合、ユーザーが入力したテキスト（日記の要約や分析のための文章など）が Anthropic の Claude API に送信されます。送信内容の取扱いについては、Anthropic の利用規約・プライバシー方針が適用されます。

保存されるデータ

日記・気分記録は端末内に保存され、当方で保管することはありません。購入情報は RevenueCat のシステムで管理されます。Claude API に送信されたデータの保管については、Anthropic の利用規約およびプライバシー方針に従います。

解析ツール

広告・アクセス解析などの解析ツールは使用していません。

第三者提供

以下の第三者に情報が提供される場合があります。

  • RevenueCat … 購入・復元処理およびサブスクリプション状態の管理のため。RevenueCat のプライバシーポリシー をご確認ください。
  • Anthropic … Claude API による AI 機能の提供のため。送信されたテキストは API の利用に必要な範囲で処理され、Anthropic のプライバシー方針に従って取り扱われます。
