# Pulse 確定仕様メモ

## 画面構成（3枚 + 設定）

- **1枚目**: 今日の状態の記録（入力画面）。5項目を1〜5で入力。記録後に2枚目へ。
- **2枚目**: 観測画面。7日/14日/30日切替。5項目それぞれのグラフを縦並びで表示。3枚目への導線「傾向分析を見る」。
- **3枚目**: 傾向分析画面。60日/90日/120日切替。AI要約（将来接続）、項目別傾向、変動、注意文。
- **設定**: 注意文・相談窓口・開発用（テストデータ投入）。振り返り導線は置かない。

## 項目（案Aで統一）

5項目: 気力・集中・疲れ・気分・眠気。全画面で共通定義（PulseMetric）を参照。直書き禁止。

## 永続化

現時点では気力・集中・疲れの3項目のみ永続化。気分・眠気はUI上は5項目表示し、2枚目では「データがありません」表示。

## トーン

静かで美しいUI。色に頼りすぎず、余白と整列で読ませる。医療用途ではない旨の注意文を維持。
