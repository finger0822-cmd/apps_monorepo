# Pulse 画面役割

| 画面 | 役割 | 期間・備考 |
|------|------|------------|
| **1枚目** | 記録（入力） | 今日の5項目を1〜5で入力。項目名長押しで説明 Bottom Sheet。入力UIに観測グラフは出さない。 |
| **2枚目** | 観測 | 7日 / 14日 / 30日切替。期間ラベル表示。5項目それぞれのグラフを縦並び（small multiples）。上部に控えめなサマリー文。「傾向分析を見る」で3枚目へ。 |
| **3枚目** | 傾向分析 | 60日 / 90日 / 120日切替。AI要約（プレースホルダ可）、項目別傾向（5項目）、変動、注意文。 |
| **設定** | 設定・注意・開発用 | 注意文、相談窓口、開発用「テストデータ投入（約120日分）」。振り返りは主機能のためホーム側メニューから。 |
